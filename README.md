# Descrição do Projeto
Um sistema de gerenciamento de tarefas em Java que roda no terminal. Permite criar, listar, editar e apagar tarefas, cada uma com título, descrição e data de conclusão.
# Como Executar/Compilar
Pré-requisitos

Java JDK 8 ou superior

Terminal/Console

1:Salve os arquivos

2:Abra o terminal na pasta raiz do projeto

3:Compile os arquivos:

javac src/model/Tarefa.java src/repository/TarefaRepository.java src/service/TarefaService.java src/appMain.java -d .

4:Execute o programa

java appMain
# Tecnologias Usadas
Java 8+

Java Collections (ArrayList)

Java Time API (LocalDate)

Scanner para entrada de dados
